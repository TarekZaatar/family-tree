Connect your Firebase project

This page is ready, but it needs a free Firebase database to store and share the tree.

  1. Create a free project at console.firebase.google.com
  2. Add a Firestore Database (production mode)
  3. Register a Web app and copy its firebaseConfig
  4. Paste that config into this file, near the top of the script, then reload

Full instructions and the security rules are in the message that came with this file.
